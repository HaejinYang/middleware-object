# 미들웨어 오브젝트 클래스

## 목적

C++에서 미들웨어(Middleware) 패턴을 쉽게 활용할 수 있는 클래스를 구현한다.

## 미들웨어 패턴이란?

미들웨어 패턴은 요청을 처리하는 일련의 파이프라인을 구성하는 방식이다.
예를 들어, 웹 프로그래밍에서 사용자 인증 → 요청 처리 → 응답 반환과 같은 단계를 거치는 방식이 이에 해당한다.
이 패턴을 사용하면 코드의 재사용성과 유연성을 높일 수 있다.
